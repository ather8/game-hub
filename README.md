# 🎮 Game Hub

A modern **React + TypeScript + Vite** application that showcases clean UI design, responsive layouts, and dynamic game content. Built as a frontend showcase project, **Game Hub** demonstrates production‑ready tooling, reusable components, and API‑driven data fetching.

---

## 🚀 Features

- **Responsive UI:** Built with **TailwindCSS** for sleek, mobile‑friendly design.  
- **Dynamic Game Data:** Fetches and displays game information from external APIs.  
- **Reusable Components:** Modular design for scalability and maintainability.  
- **Modern Tooling:** Developed with **React + TypeScript + Vite** for fast builds and type safety.  
- **Developer Experience:** ESLint + Prettier integration ensures clean, consistent code.  

---

## 🛠 Tech Stack

- **Frontend Framework:** React + TypeScript + Vite  
- **Styling:** TailwindCSS  
- **State Management:** React hooks  
- **API Integration:** RESTful endpoints for game data  
- **Tooling:** ESLint, Prettier, GitHub Actions (CI/CD ready)

## ⚡ Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/ather8/game-hub.git
   cd game-hub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
